# Таблицы

```jade
table
  thead
    tr
      td Заголовок 1
      td Заголовок 2
      td Заголовок 3
  tbody
    tr
      td Ячейка 1
      td Ячейка 2
      td Ячейка 3
    tr
      td Ячейка 1
      td Ячейка 2
      td Ячейка 3
    tr
      td Ячейка 1
      td Ячейка 2
      td Ячейка 3
```

## Теги

* `table`: Основной тег таблицы.
* `thead`: Заголовок таблицы.
* `tbody`: Содержимое таблицы.
* `tr`: Строка таблицы.
* `td`: Ячейка строки таблицы.
