# Объекты

Создание объекта:

```js
var <имя объекта> = {
    <переменная1>: <значение1>,
    <переменная2>: <значение2>,
    
    <функция1>: function (<аргумент1>, <аргумент2>, ...) {
        <тело функции>
    },
    
    <функция2>: function (<аргумент1>, <аргумент2>, ...) {
        <тело функции>
    }
};
```
